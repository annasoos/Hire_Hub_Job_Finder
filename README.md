## Implementation

### Built with

- Apollo GraphQL Server (implemented with Node & TypeScript)
- Apollo Client (implemented with React.js & TypeScript)
- JSON Web Token
- Prisma
- SQLite
- Emotions Styled Components
- Ant Design
